# text-cleaner-tool
A Python Text Cleaning and Formatting Tool with Menu-driven Features
Text Cleaner Tool (Python)

A menu-driven text cleaning tool built using Python.
It performs:

***Features***

. Lowercase conversion
. Remove punctuation
. Remove numbers
. Remove extra spaces
. Title Case formatting
. Word count
. Character count
. Sentence count
. Text preview
. Full cleaning pipeline
. Undo last action
. Reset to original text
. Save cleaned text to a file
***How to Run***
python text_cleaner_v3.py

Requirements

Python 3

re module (pre-installed)
